# Overview

Mac Digital Agency uses a premium editorial-inspired design language that balances generous whitespace, refined typography, subtle gradients, layered surfaces, and restrained motion. The experience prioritizes clarity, trust, and conversion while presenting the agency as an elite creative and technology partner.

The visual language should feel handcrafted rather than template-based. Every interaction should reinforce professionalism and confidence.

# Do's

- Use whitespace generously
- Keep gradients subtle
- Maintain consistent spacing
- Prefer purposeful animation
- Preserve strong visual hierarchy
- Use rounded geometry consistently
- Optimize for performance

---

# Don'ts

- Avoid excessive glassmorphism
- Avoid heavy shadows
- Avoid animation overload
- Avoid inconsistent spacing
- Avoid bright saturated backgrounds
- Avoid multiple competing accent colors
- Avoid dense text blocks

# Iteration Guide

The visual system is intentionally modular. Future additions such as blogs, booking flows, dashboards, multilingual support, or a CMS should reuse the existing typography, spacing, color, elevation, and component tokens to ensure a cohesive brand experience without requiring a redesign.

---

# Known Gaps

- No authenticated UI components
- No admin dashboard patterns
- No complex data visualization system
- No e-commerce checkout flows
- No dark theme specification (future enhancement)
- No design token export format (Style Dictionary/Figma variables) included in this version
