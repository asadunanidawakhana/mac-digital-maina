# Portfolio Images Directory

Drop your high-resolution portfolio images or screenshot assets inside this folder.

Recommended naming conventions to match `index.html`:
- `vertex-saas.jpg` (or `.png`)
- `nova-coffee.jpg` (or `.png`)
- `solaris-campaign.jpg` (or `.png`)
- `aura-channel.jpg` (or `.png`)
- `atlas-portal.jpg` (or `.png`)
- `aether-identity.jpg` (or `.png`)

Once you drop the images, let me know and I will update `index.html` to load them instead of the default SVGs!
